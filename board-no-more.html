Board No More

April 2019 - June 2019

Board No More is a board game business web application for the sales of popular board games. It's design was created through HTML, CSS, and JavaScript and utilizes RESTful web services for its server side logic. Board No More pulls data from a SQL database to display data and can even process orders and add those to an existing database.
