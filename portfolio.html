Website displaying my background and projects. I created it using HTML, CSS, and JavaScript. I had a responsive design so it was presentable in mobile, tablet, and desktop screens.

After doing my Board No More project, I felt confident enough to create a portfolio to display my experiences and projects. I primarily used HTML, CSS, and JavaScript to create my first rendition in a composition notebook style. I used jQuery to dynamically update the content of the page instead of using separate HTML files. You can see the github for it here:

Github

While working on my first rendition, I realized my website did not follow many web standards such as responsive design and semantic HTML. I decided to do a complete redesign and that is what you are seeing today. Try resizing your browser and seeing for yourself!
